# vue-open-weather
Open API - 날씨 공공데이터를 이용한 날씨 확인 웹 사이트 구현. SPA, Vue.js, Spring Boot 등 최신기술 습득


## Project Setup
```
cd frontend
npm install
```

### Compile & Hot reload for development
```
npm run serve
```


## Weather API
OpenWeatherMap 사용

```
url = api.openweathermap.org/data/2.5/weather?q={city name}&appid={API key}
```
